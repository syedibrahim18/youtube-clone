# YouTube Clone (HTML & CSS)

This is a **YouTube Homepage Clone** built using **HTML and CSS** to practice frontend layout and styling.

## 🚀 Features

* Responsive YouTube-style layout
* Header with search bar
* Sidebar navigation
* Video grid layout
* Channel profile pictures
* Video thumbnails with duration
* Clean UI using CSS Flexbox & Grid

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts

## 📷 Preview

YouTube homepage inspired design with video cards and sidebar.
